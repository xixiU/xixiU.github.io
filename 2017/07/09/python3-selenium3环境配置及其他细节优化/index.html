发表于 2017-07-09 | 分类于 python爬虫
ubuntu下python3+selenium3环境配置

之前在windows使用的firefox和chrome都是绿色版，配置的selenium3一直没有成功，于是尝试着在Linux下配置，成功配置了！

准备

OS: Linux ubuntu 16.04
python3:Python 3.5.2
Firefox默认安装

#下载selenium3#

  1. 确定操作系统架构
    打开终端，输入sudo uname -p

    • 若返回i686，证明为32为操作系统
    • 若返回x86_64,证明为64位操作系统

  2. 安装selenium3

    安装geckodriver 记住要在解压之后的父目录操作
    cp geckodriver /usr/local/bin/

  4. 安装xvfb
    sudo apt-get install xvfb

  5. 安装pyvirtualdisplay

  6. 测试 保存为test.py

然后运行

  • 定位到之前geckodriver的目录，设置权限为-rwxr-xr-x
    即sudo chmod 755 geckodriver
