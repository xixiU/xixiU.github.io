ubuntu清理垃圾

  1. 清理apt载安装软件缓存文件
    $sudo apt-get clean

  2. 清理期软件包
    $sudo apt-get autoclean

  3. 自卸载没软件依赖关系软件
    $sudo apt-get autoremove
